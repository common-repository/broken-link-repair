Broken Links Repair Plugin disables the bad links in your content immediately upon detection by Hexometer.com scanner.

== Description ==

Broken Links Repair Plugin will get the broken links list from your Hexometer dashboard and will disable (or “unlink”) them when the page is loaded.


= Key features =

Key features of Broken Links Repair Plugin by Hexometer include:

* Disables the 404 links in the page content.
* You can enable and disable this function from plugin page.
* More info about [Broken Links Repair](https://hexometer.com/broken-links-repair)

For Any help, please [contact us](https://hexometer.com/contact).


== Installation ==

After Installation you have to verify your account by submitting the verification code from Hexometer.com account:

A. For new users - Create an account at [Hexometer.com](https://hexometer.com) add your website to monitoring service and copy the verification code from the settings page.

B. If you already monitor this website by Hexometer, then copy the verification code from the settings page.

== Frequently Asked Questions ==

= What are the Limits? =

Currently, free accounts are limited up to 10,000 URLs, if your website has more URLs then you have to upgrade to the appropriate package, otherwise, the system will scan only the mentioned amount of links.

= Is My Content Safe With This Plugin? =

Yes! Your content is absolutely secure. We do not edit your content, the plugin disables the broken link's "a href" tag from the content before the page loads. And you can switch ON/OFF anytime from your plugin page.
